## How To Use

1. Duplicate this folder into `launch-kits/{partner-name}/`.
2. Replace placeholder fields with partner-specific context.
3. Create audience-specific one-pagers, scripts, handouts, trackers, and site prompts as needed.
4. Keep reusable content in `templates/` or top-level shared folders.
5. Keep partner-specific content inside the partner launch-kit folder.

## Partner Launch Kit Goal

A launch kit should help a partner:

- Understand their strongest lanes
- Know who to contact first
- Start referral conversations
- Track outreach and follow-up
- Submit applications responsibly
- Review weekly activity
- Improve assets based on field feedback
