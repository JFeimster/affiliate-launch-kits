# 30-Day Launch Plan Template

```markdown

- Improve scripts based on feedback.

### Objectives

- Review what worked.
- Identify repeatable audiences.
- Refine assets.
- Plan the next 30 days.

### Actions

| Action | Owner | Due Date | Status |
|---|---|---|---|
| Complete weekly review | [Owner] | YYYY-MM-DD | Draft |
| Update scripts from field feedback | [Owner] | YYYY-MM-DD | Draft |
| Identify top referral source lane | [Owner] | YYYY-MM-DD | Draft |
| Draft month-2 scale plan | [Owner] | YYYY-MM-DD | Draft |

1. Which audience responded best?
2. Which script produced the clearest conversations?
3. What objections showed up repeatedly?
4. What assets are missing?
5. What should change next week?

- Keep language educational and relationship-first.
```
