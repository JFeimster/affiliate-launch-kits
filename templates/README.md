# Templates

Reusable templates for partner enablement assets live here.

Use these templates to speed up new partner kit creation without duplicating source content across random docs, dashboards, and platform drafts.

## Template Index

| Template | Path | Purpose |
|---|---|---|
| Launch Kit Template | `launch-kit-template/` | Starter folder for a new partner launch kit |
| Partner Profile Template | `launch-kit-template/partner-profile.md` | Defines partner positioning, lanes, referral sources, and launch priorities |
| Launch Kit Index Template | `launch-kit-template/launch-kit-index.md` | Creates the table of contents and operating map for a partner launch kit |
| 30-Day Launch Plan Template | `launch-kit-template/launch-plan/30-day-launch-plan.md` | Creates the first-month partner launch sprint |
| One-Pager Template | `one-pager-template.md` | Creates concise referral, niche, or partner one-pagers |
| Weekly Tracker Template | `weekly-tracker-template.md` | Tracks activity, referrals, applications, follow-ups, and weekly lessons |
| Outreach Script Pack Template | `outreach-script-pack-template.md` | Creates partner outreach and follow-up scripts |
| Website Prompt Template | `website-prompt-template.md` | Creates landing page, Wix, Flash-UI, or static site prompts |
| Handout Print Template | `handout-print-template.md` | Creates printable handouts and PDF-ready resources |
| Booking Event Template | `booking-event-template.md` | Creates booking pages, recurring meetings, and partner review events |
| Notion Database Prompt Template | `notion-database-prompt-template.md` | Creates Notion database specs and dashboard prompts |

## How To Use

1. Start with `launch-kit-template/` for a new partner.
2. Duplicate only the files needed for the partner's launch stage.
3. Place partner-specific versions under `launch-kits/{partner-name}/`.
4. Keep reusable patterns in `templates/`.
5. Use GitHub as the source-of-truth before adapting content into Notion, Wix, Google Drive, HubSpot, Gmail, Canva, or other platforms.
