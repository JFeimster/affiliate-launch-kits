# Affiliate Launch Kits

This repository stores source files, prompt chains, launch plans, printable handout copy, outreach scripts, and microsite prompts for Moonshine Capital affiliate partners.

The goal is to create reusable, version-controlled launch kits that help new funding partners move from onboarding to conversations, applications, referrals, and funded deals.

## Core Workflow

1. Capture source material from calls, transcripts, and onboarding notes.
2. Store structured notes in `/source-notes`.
3. Generate launch plan, tracker, handouts, scripts, and site prompts.
4. Store final markdown source files in the affiliate folder.
5. Link final assets to Notion, Google Drive, Wix, and HubSpot.
