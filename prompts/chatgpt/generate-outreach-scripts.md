# Generate Outreach Scripts Prompt

Use this prompt to generate partner outreach and follow-up scripts.

## Asset Status

- Status: Ready
- Owner: Moonshine Capital
- Audience: Internal Operator
- Platform: ChatGPT
- Last reviewed: 2026-05-03

## Objective

Create a practical script pack for referral partner conversations, warm introductions, direct outreach, follow-ups, and referral asks.

## Inputs Needed

- Partner name
- Target audience
- Relationship context
- Offer/resource angle
- CTA or next step
- Known objections

## Prompt

```text
Create an outreach script pack for a Moonshine Capital partner.

Partner: [Partner Name]
Target audience: [Audience]
Relationship context: [Cold / Warm / Existing network / Referral source]
Offer/resource angle: [Angle]
CTA or next step: [CTA]
Known objections: [Objections]

Create scripts for:
1. Short DM
2. Email
3. Phone opener
4. LinkedIn message
5. First follow-up
6. Second follow-up
7. Referral ask
8. Objection handling
9. Soft reactivation message

The scripts should be direct, professional, relationship-first, and easy to customize.

Do not overhype. Do not position banks, accountants, attorneys, or other professionals as enemies. Position the partner as a useful resource when someone needs a responsible next step.

Do not guarantee approval, terms, amounts, rates, income, or timelines.

End with:
- suggested GitHub path
- suggested Notion status update
- suggested export filename
- next recommended deliverable
```

## Suggested GitHub Output Path

```text
launch-kits/{partner-name}/scripts/{audience}-outreach-scripts.md
```
